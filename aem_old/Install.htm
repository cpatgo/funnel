AwebDesk Email Marketing Software 7.1.0 Installation Guide

Thank you for choosing Email Marketing Software from AwebDesk.

This document will assist you with the installation process. If you have questions or need assistance at any time please visit our support center.

Follow these steps to install AwebDesk Email Marketing Software.

  1. Unzip the downloaded ZIP file.

  2. Rename Rename_to-serialkey.php to serialkey.php in /cache/ directory and Enter the license key which you received on purchasing in this file. Also update the install url in the same file.

  3. Rename Rename_to-config.inc.php to config.inc.php in /upload/manage/ directory

  4. Rename Rename_to-config_ex.inc.php to config_ex.inc.php in /upload/manage/ directory

  5. Upload the contents of package /upload/ to your web server where you want to install the product.In this example we assume you are installing on /AEM/ directory.

  6. Adjust the file permissions on the following files:

    • If installing on a linux server you should CHMOD the following files/folders to 777. Which is full read/write permissions.
    • If installing on a Windows server you should ensure IIS has full control / full permissions for the following files/folders.

      • /AEM/manage/config.inc.php
      • /AEM/manage/config_ex.inc.php
      • /AEM/cache/key.php
      • /AEM/cache/
      • /AEM/images/
      • /AEM/images/manage/

        • All files and folders inside:

      • /AEM/cache/public/
      • /AEM/cache/admin/
      • /AEM/cache/manage/

      • If you find that some of the folders listed above are missing, this may have been caused by your compression program. All that you need to do is add the folders to their respected directories

  7. Run the web based installer by opening the following file (which is located on your web server) in your web browser
    • /AEM/manage/install.php
    • If something goes wrong or in case you get any license error after installation, please verify that /cache/ directory and all contents inside it is writable(777 permissions).Also try deleting key.php file from /AEM/cache/ on server and refresh the page.

  8. Refresh the page twice on first install screen. Follow the onscreen instructions. That completes your setup! You may now login to your software application.


After installing the software, you should also...

  • Set up a cron job or scheduled task on your server, which runs the manage/cron.php file every five minutes. This will ensure that all scheduled processes (such as scheduled campaigns and autoresponders) start at the correct times, and that large campaigns or imports do not stall, and will keep running even if you are not logged in to the software. For a basic overview regarding cron jobs, refer to this article: Setting Up A Cron Job

  • Edit your mail sending methods and throttling settings, or add multiple SMTP servers to send with. To add or edit your mail settings, click the Settings link in the upper right corner of the software interface, and scroll to the Mail Settings section on that page. Refer to this article on our site for help configuring mail settings: Mail Sending Options

  • Configure bounce management, to ensure that any emails that bounce back to you are properly tracked, and that invalid email addresses are automatically removed from the system. Click the Lists tab, and then click Bounce Settings to edit or add a new bounce management settings. The following article explains how to set up Bounce Management: Setting Up Bounce Management
